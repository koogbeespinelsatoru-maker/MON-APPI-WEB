  POUR TESTER L'APPLICATION MAINTENANT :
════════════════════════════════════════════════════════════════

OPTION A - The Easy Way (Recommandé) 👈
───────────────────────────────────────
1. Double-cliquez sur : start.bat
2. Sélectionnez option [1] : Mode Complet
3. Attendez ~30 secondes le démarrage
4. Ouvrir http://localhost:5173 dans navigateur

✨ C'est tout ! L'app démarre avec Backend + Frontend


OPTION B - Manual (Pour les pros)
───────────────────────────────────

Terminal 1 - Backend :
  cd backend
  npm start
  → Vérifier : "🚀 Serveur démarré sur http://localhost:5000"

Terminal 2 - Frontend :
  cd frontend
  npm run dev
  → Ouvrir : http://localhost:5173

SI RIEN NE S'AFFICHE (Troubleshooting rapide)
════════════════════════════════════════════════════════════════

❌ "Cannot find module" ?
   └─ npm install dans /backend + /frontend

❌ "Photos ne s'affichent pas" ?
   └─ Backend doit être sur localhost:5000
   └─ F5 (refresh) navigateur

❌ "Erreur CORS" ?
   └─ Vérifier frontend sur localhost:5173

❌ "Likes ne fonctionnent toujours pas" ?
   └─ Hard refresh : Ctrl+Shift+R
   └─ Vérifier console (F12) pour erreurs
