SCRIPTS DE DÉMARRAGE
────────────────────────────────────────────────────────────────────────

🔧 start.bat (Windows) - AMÉLIORÉ
   ├─ Usage : Double-cliquez ou `start.bat [option]`
   ├─ Options :
   │  ├─ (aucun) ............ Menu interactif
   │  ├─ backend .......... Backend seul
   │  ├─ frontend ......... Frontend seul
   │  ├─ all .............. Les deux (RECOMMANDÉ)
   │  └─ -help ............ Affiche aide
   └─ Features :
      ├─ Vérification prérequis (Node.js, dossiers)
      ├─ Installation auto npm si besoin
      ├─ Lancement 2 terminaux séparés
      └─ Affichage URLs après démarrage

🔧 start.sh (Linux/Mac)
   ├─ Usage : `bash start.sh [option]`
   ├─ Options : Mêmes que start.bat
   └─ Features : Mêmes + compatible système Unix
